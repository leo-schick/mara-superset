Mara Superset documentation
===========================

Welcome to Mara Superset's documentation. This extension to the `Mara Framework <https://github.com/mara>`_
synchronizes data models to `Apache Superset <https://superset.apache.org>`_.
It relies on the `Mara Schema <https://github.com/mara/mara-schema>`_ module.

Get started with :ref:`Installation<installation>` and then get an overview with the :ref:`Quickstart`.
